# Pointers

This section focuses on understanding pointers in C, which are essential for low-level programming and memory manipulation.

## 🎯 Objective
Gain a solid understanding of how pointers work and how to use them safely.

## 📚 Topics covered
- pointer basics (*, &)
- pointer arithmetic
- arrays and pointers
- passing pointers to functions

## 🧠 What I’m learning
- how memory is accessed and modified
- how to avoid common mistakes (invalid access, undefined behavior)
- how pointers interact with arrays and functions

## ⚠️ Challenges
- segmentation faults due to invalid memory access
- confusion between pointer and value
- debugging pointer-related issues
