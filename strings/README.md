# Strings

This section focuses on string manipulation in C, which requires careful handling of memory and character arrays.

## 🎯 Objective
Learn how to work with strings efficiently and safely in C.

## 📚 Topics covered
- character arrays
- standard string functions (strlen, strcmp, strcpy)
- string parsing and tokenization
- manual string manipulation

## 🧠 What I’m learning
- how strings are represented in memory
- how to avoid buffer overflows
- how to manipulate strings without introducing bugs

## ⚠️ Challenges
- off-by-one errors
- forgetting null terminators
- modifying immutable strings
